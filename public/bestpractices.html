Comment section

In order for this discussion board to function properly we have instated some rules, the rules are as follows:

1. Before creating a new thread or posting a comment in the comment section, a user should make sure that his specific topic or comment has not been posted on the board already.

2. Be polite and constructive whilst providing comments

3. Please be fittingly elaborate in your comment. Although comments should be concise and succinct, comments such as “I agree” are not sufficient.

4. Only post comments related to the topic.

5. Do not dominate the conversation as this blocks knowledge construction.

In order for this discussion board to function properly and to get the best use out of the functionality we have some tips to give:

1. When writing feedback be sure to be personal, frequent, immediate, balanced, specific and positive in commenting/reacting

2. When writing a top-level comment of a specific type be sure to label the comment accordingly

3. When writing a question comment be sure to write out the question explicitly at the start of the comment
